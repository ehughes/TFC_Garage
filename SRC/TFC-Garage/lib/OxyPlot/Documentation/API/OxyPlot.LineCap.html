LineCap Enumeration

Specifies the line cap type.

Namespace: OxyPlot
Assembly: OxyPlot (in OxyPlot.dll)

Syntax

public enum LineCap : Enum

Members

Name Description
Butt Butt cap. The stroke is squared off at the endpoint of the path. There is no projection beyond the end of the path.
ProjectingSquare Projecting square cap. The stroke continues beyond the endpoint of the path for a distance equal to half the line width and is squared off.
Round Round cap. A semicircular arc with a diameter equal to the line width is drawn around the endpoint and filled in.
