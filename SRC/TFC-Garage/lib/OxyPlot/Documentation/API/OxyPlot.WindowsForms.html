OxyPlot.WindowsForms Namespace

The OxyPlot.WindowsForms namespace contains controls for Windows Forms and a bitmap exporter.

Classes

Name Description
ConverterExtensions Extension method used to convert to/from Windows/Windows.Media classes.
GraphicsRenderContext The graphics render context.
Plot Represents a control that displays a plot.
PlotModelExtensions Provides extension methods to the PlotModel.
PngExporter Provides functionality to export plots to png.
SvgExporter Provides functionality to export plots to scalable vector graphics using System.Drawing.Graphics for text measuring.
