OxyPlot.Series Namespace

The OxyPlot.Series namespace contains the series and related types.

Classes

Name Description
AreaSeries Represents an area series that fills the polygon defined by two sets of points or one set of points and a constant.
BarItem Represents an item used in the BarSeries.
BarItemBase Represents an item used in the BarSeriesBase.
BarSeries Represents a series for clustered or stacked bar charts.
BarSeriesBase Base class for BarSeries and ColumnSeries.
BarSeriesBase<T> Generic base class that provides common properties and methods for the BarSeries and ColumnSeries.
BoxPlotSeries Represents a series for box plots.
CandleStickSeries Represents a series for candlestick charts.
CategorizedItem Represents an item in a CategorizedSeries.
CategorizedSeries Base class for series where the items are categorized.
ColumnItem Represents an item used in the ColumnSeries.
ColumnSeries Represents a series for clustered or stacked column charts.
ContourSeries Represents a series that renders contours.
DataPointSeries Provides an abstract base class for series that contain a collection of IDataPoints.
ErrorColumnItem Represents an item used in the ErrorColumnSeries.
ErrorColumnSeries Represents a series for clustered or stacked column charts with an error value.
FunctionSeries Represents a line series that generates its dataset from a function.
HeatMapSeries Represents a heat map.
HighLowItem Represents an item in a HighLowSeries.
HighLowSeries Represents a series for high-low plots.
IntervalBarItem Represents an item in an IntervalBarSeries.
IntervalBarSeries Represents a series for bar charts defined by to/from values.
ItemsSeries Abstract base class for series that can contain items.
LineSeries Represents a line series.
PieSeries Represents a series for pie/circle/doughnut charts.
PieSlice Represent a slice of a PieSeries.
RectangleBarItem Represents a rectangle item in a RectangleBarSeries.
RectangleBarSeries Represents a series for bar charts where the bars are defined by rectangles.
ScatterPoint Represents a point in a ScatterSeries.
ScatterSeries Represents a series for scatter plots.
Series Provides an abstract base class for plot series.
StairStepSeries Represents a series for stair step graphs.
StemSeries Represents a series that plots discrete data in a stem plot.
TornadoBarItem Represents an item for the TornadoBarSeries.
TornadoBarSeries Represents a series that can be used to create tornado plots.
TwoColorLineSeries Represents a two-color line series.
XYAxisSeries Provides an abstract base class for series that are related to an X-axis and a Y-axis.
