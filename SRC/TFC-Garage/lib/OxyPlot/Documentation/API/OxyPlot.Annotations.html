OxyPlot.Annotations Namespace

The OxyPlot.Axes namespace contains the annotations and related types.

Classes

Name Description
Annotation Provides an abstract base class for annotations.
ArrowAnnotation Represents an arrow annotation.
EllipseAnnotation Represents an ellipse annotation.
FunctionAnnotation Represents a line annotation defined
ImageAnnotation Represents a text annotation.
LineAnnotation Represents a line annotation.
PathAnnotation Represents all annotations who are paths such as lines, functions or polylines
PolygonAnnotation Represents a polygon annotation.
PolylineAnnotation Represents a polyline annotation.
RectangleAnnotation Represents a rectangle annotation.
TextAnnotation Represents a text annotation.
TextualAnnotation Provides an abstract base class for annotations that contains text.
TileMapAnnotation Provides a tile map annotation.

Name Description
