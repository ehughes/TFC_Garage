OxyPlot.OpenXml Namespace

The OxyPlot.OpenXml namespace contains functionality to generate OpenXml (Word) documents from the OxyPlot.Reporting report model.

Classes

Name Description
WordDocumentReportWriter Provides a report writer for Word/OpenXML (.docx) output using OpenXML SDK 2.0.
