OxyPlot namespaces

The OxyPlot solution provides plotting functionality for Windows store apps, WPF, Windows forms, Silverlight and Mono based applications.

Name Description
OxyPlot The OxyPlot namespace contains the platform independent classes of the library.
OxyPlot.Annotations The OxyPlot.Axes namespace contains the annotations and related types.
OxyPlot.Axes The OxyPlot.Axes namespace contains classes that represents the axes of the plots.
OxyPlot.OpenXml The OxyPlot.OpenXml namespace contains functionality to generate OpenXml (Word) documents from the OxyPlot.Reporting report model.
OxyPlot.Pdf The OxyPlot.Pdf namespace contains functionality to export plots and reports to .pdf files.
OxyPlot.Reporting The OxyPlot.Reporting namespace contains a simple report model.
OxyPlot.Series The OxyPlot.Series namespace contains the series and related types.
OxyPlot.WindowsForms The OxyPlot.WindowsForms namespace contains controls for Windows Forms and a bitmap exporter.
OxyPlot.Wpf The OxyPlot.Wpf namespace contains WPF controls and PNG export functionality based on WPF.
OxyPlot.Xps The OxyPlot.Xps namespace contains functionality to export plots and reports to .xps files.
